Основы CSS

CSS позволяет управлять цветами, шрифтами и расположением элементов на странице.

Чтобы изменить стиль элементов, используют селекторы, а затем задают свойства с значениями.

Для интерактивности можно использовать псевдоклассы, а для адаптивности — @media.

Блочная модель (box model) определяет размеры и отступы элементов: margin, padding, border, width/height.
